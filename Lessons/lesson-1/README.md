# Lesson 1.0
## Your First Application is "Hello, World"
**Objectives**
1. Students can display "Messages" on monitor.
2. Students can run Python program by theirselves.


**Instruction**
1. Open your Editor
2. Create "New File" as **hello.py**
3. type below
```python
print("Hello, World")
```
4. Press F5, for run the program

OUTPUT
```python
Hello, World
```
